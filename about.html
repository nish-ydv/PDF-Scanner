About PDF Scanner

PDF Scanner is a simple, free web tool that helps you work with PDF files directly in your browser. You can merge PDFs, convert images to PDF, and reduce file sizes without installing any software.

All file processing happens locally in your browser using JavaScript. Your files are never uploaded to a server, which means your data stays private and secure.

This tool is useful for students, professionals, and anyone who needs quick PDF tasks without signing up or paying.

PDF Scanner is a frontend-only project built for learning and demonstration purposes. Some advanced features may be limited compared to paid tools.
